1. Nmap Scanner


Supports private IPv4 addresses in the following formats:

- 10.0.0.0/8 IP addresses: 10.0.0.0 – 10.255.255.255
- 172.16.0.0/12 IP addresses: 172.16.0.0 – 172.31.255.255
- 192.168.0.0/16 IP addresses: 192.168.0.0 – 192.168.255.255

It also supports optional CIDR or hyphen notation at the end of the IP address string (e.g., /24 or -16).
